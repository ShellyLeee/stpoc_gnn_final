# Biological Prior Information

This directory contains additional biological features and priors.

## Cell Abundance Features

Generated by cell2location (deconvolution of spatial transcriptomics):

- `cell2location_predicted_cell_abundance_mean_train.csv`
- `cell2location_predicted_cell_abundance_mean_valid.csv`  
- `cell2location_predicted_cell_abundance_mean_test.csv`

## Protein Co-expression Matrix

`protein_coexpression_matrix.txt`

**Source**: STRING database protein-protein interaction scores

## Usage in Config

Enable these features in `cfgs/gnn.yaml`:
```yaml
data:
  use_cell_features: true
  cell_features_dir: "data/bio_info/"
  cell_abundance_mean_train: "cell2location_predicted_cell_abundance_mean_train.csv"
  # ...

coexpression:
  enabled: true
  matrix_path: "data/bio_info/protein_coexpression_matrix.txt"
```
